# Православная школа — Vue/Vite

## Запуск
```bash
npm install
npm run dev
```

## Структура
- `src/pages/` — отдельные страницы сайта.
- `src/components/` — переиспользуемые блоки: шапка, подвал, карточки.
- `src/assets/images/` — изображения макета.
- `src/styles/main.css` — общая стилизация под макет.
- `src/data/news.js` — данные новостей.
